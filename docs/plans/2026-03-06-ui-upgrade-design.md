# UI升级设计方案：极简现代风 + 温暖奶油风

## 设计目标

将智能合同审核系统前端从通用风格升级为**优雅专业的法律工作台**，采用极简现代设计语言搭配温暖奶油色调。

---

## 1. 配色方案

| 用途 | 颜色代码 | 说明 |
|------|----------|------|
| 主背景 | `#F5F0E8` | 暖米白，柔和护眼 |
| 卡片背景 | `#FFFDF9` | 象牙白 |
| 强调色 | `#D4A574` | 暖棕色/驼色，用于按钮、图标 |
| 高风险 | `#E07A5F` | 陶土红 |
| 中风险 | `#E9B44C` | 琥珀黄 |

| 文字主色 | `#3D405B` | 深灰蓝 |
| 文字次要 | `#6B705C` | 橄榄灰 |
| 边框色 | `#E8E2D9` | 暖灰 |

---

## 2. 字体方案

- **标题字体**：Noto Serif SC（典雅中文衬线体）
- **正文字体**：Noto Sans SC（清晰无衬线）
- **数据字体**：JetBrains Mono（等宽数字）

引入方式：
```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@400;600;700&family=Noto+Sans+SC:wght@300;400;500;600&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet">
```

---

## 3. 组件设计

### 3.1 侧边栏

- 毛玻璃效果（backdrop-filter: blur）
- 暖色渐变标题区
- 悬停时按钮微微上浮

### 3.2 统计卡片

- 奶油底色 + 彩色左边框
- 圆角增加到 16px
- 轻微阴影：0 4px 20px rgba(0,0,0,0.06)

### 3.3 风险标签

- 柔和边框（1px solid）
- 背景色为原色的 15% 透明度
- 悬停时边框变实

### 3.4 上传区域

- 虚线边框改为实线
- 添加图标 + 引导文字
- 拖拽时边框变色 + 背景微亮

### 3.5 Tab 切换

- 底部移动指示器（animated underline）
- 悬停时文字颜色变化
- 选中态底部色条

### 3.6 条款卡片

- 悬停时：translateY(-2px) + 阴影加深
- 选中态：左边框加粗

---

## 4. 动效设计

### 4.1 页面加载动画

```css
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
```

- 卡片依次延迟 0.1s 出现
- 整体过渡时间 0.4s

### 4.2 交互动效

- 按钮悬停：scale(1.02)，0.2s ease
- 卡片悬停：translateY(-2px)，0.3s ease
- Tab 指示器：滑动手感，0.3s ease
