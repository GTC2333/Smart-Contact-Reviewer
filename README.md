# 🧠 智能合同审核系统

## 系统简介

本系统利用 AI 技术对商业合同进行**自动化审核**，识别潜在风险并生成修改建议。支持 **PDF / Word / TXT** 格式的合同文件。

---

## 快速开始

### 启动系统

```bash
# 方式一：一键启动（推荐）
./start_all.sh

# 方式二：分别启动
# 终端1：启动后端
PYTHONPATH=. uvicorn api.server:app --host 127.0.0.1 --port 8000 --reload

# 终端2：启动前端
streamlit run frontend/app.py
```

### 访问地址

- 前端界面：http://localhost:8501
- 后端 API：http://localhost:8000
- API 文档：http://localhost:8000/docs

---

## 使用指南

### 1. 上传合同

在首页点击上传区域或拖拽文件，支持：
- `.txt` 文本文件
- `.pdf` PDF 文档
- `.docx` Word 文档

### 2. 开始审核

上传合同后，系统会自动开始审核分析。您可以：
- 实时查看审核进度
- 对照阅读合同原文
- 等待审核完成

### 3. 查看结果

审核完成后，显示：
- **风险统计**：高/中/低风险数量
- **风险详情**：每项风险的问题描述、严重程度、法律依据
- **修改建议**：针对每项风险的修改建议
- **合同原文**：完整的合同文本（可复制）

### 4. 导出报告

支持导出为 PDF 或 Word 格式的审核报告。

### 5. 历史记录

所有审核记录会自动保存，可通过"历史记录"查看：
- 重新查看审核结果
- 导出历史报告
- 删除不需要的记录

## 配置说明

### LLM 模型配置

编辑 `config/settings.yaml`：

```yaml
llm:
  provider: "deepseek"  # 可选: openai, anthropic, gemini, deepseek
  api_key: "your-api-key"
```

### API 端口配置

```yaml
backend_api:
  host: "127.0.0.1"
  port: 8000
```

## 常见问题

### 1. 上传文件失败
- 检查文件格式是否支持（.txt/.pdf/.docx）
- 确认文件大小适中

### 2. 审核时间过长
- 合同内容较多时需要更长时间
- 检查网络连接是否正常
- 确认 API Key 配置正确

### 3. 风险识别不准确
- AI 识别基于文本内容，建议人工复核
- 可在"历史记录"中查看和对比结果
